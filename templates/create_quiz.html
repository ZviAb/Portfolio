Build your quiz with multiple choice questions

Upload a JSON or Excel file to automatically populate your quiz with questions. This will override manual question entry.

JSON Format Example:

{
  "questions": [
    {
      "text": "What is the capital of France?",
      "option_a": "London",
      "option_b": "Berlin",
      "option_c": "Paris",
      "option_d": "Madrid",
      "correct_option": "C"
    }
  ]
}

Excel Format Example:

Create an Excel file with these column headers:

text option_a option_b option_c option_d correct_option
